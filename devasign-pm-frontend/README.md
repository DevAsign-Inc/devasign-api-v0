# Devasign Frontend

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/pages/api-reference/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying the files in the `src` directory. The pages auto-update as you edit the files.

## Deployment

### Build for Production

To build the application for production, run:

```bash
npm run build
```

### Start Production Server

To start the production server after building:

```bash
npm run start
```

You can deploy this application to any hosting provider that supports Node.js applications.
